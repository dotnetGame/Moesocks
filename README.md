Moesocks
====

A proxy server / client written in C#.

使用C#编写的代理服务器和客户端。

## Screenshots
![image](https://github.com/sunnycase/Moesocks/raw/master/screenshots/2017-7-11.png)

## Feature ##
- Http proxy
- Socks5 local server
- TLS 1.2 Connection
- Certificate based authentication
- Random data obfuscation
